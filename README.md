# .dotfiles

Must be cloned in `~/.dotfiles`

Install:

```
sudo apt-get install git
git clone git@github.com/avaly/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

Local overrides can be made in `~/.dotfiles.local`

Uninstall:

```
./uninstall.sh
```
